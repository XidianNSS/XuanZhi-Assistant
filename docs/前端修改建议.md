# 前端修改建议

## 一、管理员面板（需要 role === 'admin'）

admin 用户登录后，在侧边栏或顶部增加"管理"入口，展示以下内容：

### 1.1 概览统计
调用 `GET /api/admin/stats` 获取：用户数、Agent 数、任务总数/运行中/已完成/失败数、Gateway 连接状态。

### 1.2 用户列表
调用 `GET /api/admin/users`，展示表格：邮箱、角色、注册时间。无需增删改。

### 1.3 Agent 列表
调用 `GET /api/admin/agents`，展示每个 Agent 的名称、所属用户、运行状态（offline/idle/running/error）、sessionKey、gatewayAgentId。

### 1.4 任务列表
调用 `GET /api/admin/tasks`，展示所有用户的任务：标题、所属用户、意图、状态、创建时间。

### 1.5 Gateway Agent 状态
调用 `GET /api/admin/gateway/agents`，展示 Gateway 上真实运行的 agent 进程信息。

---

## 二、技能市场（Skill Marketplace）

一个"技能商店"页面，用户可以浏览、搜索、查看技能详情，并安装到自己的 Agent。

### 2.1 技能目录浏览
调用 `GET /api/skills/catalog?q=关键词&limit=20` 获取技能列表和分类目录（bins）。
- 不带 `q` 时返回全量列表
- 带 `q` 时搜索匹配的技能
- `bins` 用于左侧分类导航

### 2.2 技能详情
调用 `GET /api/skills/catalog/:slug`，展示技能的完整描述、版本、作者、标签、安装状态。

### 2.3 安装技能到 Agent
调用 `POST /api/agents/:agentId/skills/install`，body: `{ slug, version? }`。
安装按钮应显示加载状态，安装完成后刷新该 Agent 的技能列表。

---

## 三、Agent 技能管理

每个用户在自己的 Agent 设置中可以看到已安装的技能。

### 3.1 查看已安装技能
调用 `GET /api/agents/:agentId/skills`，列出该 Agent 当前已安装的技能及启用状态。

### 3.2 启用/禁用技能
调用 `PATCH /api/agents/:agentId/skills/:skillKey`，body: `{ enabled: true/false }`。
每个技能旁边有开关按钮。

### 3.3 配置技能参数
同一个 `PATCH` 接口，body 可包含 `{ apiKey, env: { KEY: "value" } }`。
需要 API Key 的技能应显示配置入口（如输入框），在调用时传入 `apiKey` 字段。

---

## 四、API 端点汇总

| 端点 | 方法 | 用途 | 权限 |
|------|------|------|------|
| `/api/admin/stats` | GET | 概览统计 | admin |
| `/api/admin/users` | GET | 用户列表 | admin |
| `/api/admin/agents` | GET | Agent 列表 | admin |
| `/api/admin/tasks` | GET | 全部任务 | admin |
| `/api/admin/agents/:agentId/tasks` | GET | 某 Agent 的任务 | admin |
| `/api/admin/gateway/agents` | GET | Gateway Agent 进程 | admin |
| `/api/skills/catalog` | GET | 技能市场（搜索+分类） | 已登录 |
| `/api/skills/catalog/:slug` | GET | 技能详情 | 已登录 |
| `/api/agents/:agentId/skills` | GET | Agent 已安装技能 | owner/admin |
| `/api/agents/:agentId/skills/install` | POST | 安装技能 | owner/admin |
| `/api/agents/:agentId/skills/:skillKey` | PATCH | 启用/禁用/配置 | owner/admin |

---

## 五、数据流

```
技能市场页面
  ├── GET /api/skills/catalog  → 浏览/搜索技能
  ├── GET /api/skills/catalog/:slug → 查看技能详情
  └── POST /api/agents/:agentId/skills/install → 安装到我的 Agent

Agent 设置页面
  ├── GET /api/agents/:agentId/skills → 已安装列表
  └── PATCH /api/agents/:agentId/skills/:skillKey → 开关/配置

管理员面板
  ├── GET /api/admin/stats     → 概览
  ├── GET /api/admin/users     → 用户管理
  ├── GET /api/admin/agents    → Agent 管理
  └── GET /api/admin/tasks     → 任务管理
```
